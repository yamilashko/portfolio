# Portfolio (учебный проект по Веб-технологиям)

Личный сайт-портфолио, созданный для представления информации обо мне, моих навыках, учебном и проектном опыте.

## Описание проекта

Этот репозиторий содержит многостраничный сайт-портфолио. На сайте собрана основная информация обо мне, моём профессиональном пути, навыках, проектах и контактах.

Сайт может использоваться как онлайн-визитка для стажировок, учебных проектов и откликов на вакансии.

## Структура сайта

```text
portfolio/
│
├── index.html       # Главная страница
├── about.html       # Информация обо мне
├── skills.html      # Навыки
├── projects.html    # Проекты
├── journey.html     # Образовательный и профессиональный путь
├── contact.html     # Контакты
├── styles.css       # Стили сайта
├── script.js        # JavaScript для интерактивных элементов
└── images_.zip      # Архив с изображениями
